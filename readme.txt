== Description ==



Bu eklenti sayesinde Wordpress blogunuzda Kral FM Radyo kurulmaktadır.



== Installation ==


Eklentiyi yükleyip aktive ettikten sonra Wordpress Admin panelinden Görünüm > Bileşenler içinde 'Kral FM Radyo' bileşenini bulup yan menüde istediğiniz bölüme sürükleyip yerleştirin ve kaydedin..



== Frequently Asked Questions ==


Kral FM'i dinleyebilmeniz için Admin panelinden Görünüm > Bileşenler içinde seçtiğiniz alana bileşeni sürükleyip bırakmanız yeterlidir.
